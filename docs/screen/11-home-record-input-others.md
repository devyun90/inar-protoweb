| 항목 | 내용 |
|------|------|
| 위치 | FAB 빠른 기록 바텀시트 / 기록 서브탭 > 카테고리별 입력 |

## 1. 레이아웃

```

│ [✕]   수면 기록                    [저장]    │

│  시작   [ 2025-06-30  21:00 ▾ ]             │
│  종료   [ 2025-06-30  23:10 ▾ ]  (선택)     │
│         └ 진행 중 [ 토글 ] (종료 비움)       │
```

## 3. 상태 분기

### 기본 (입력)
- 카테고리별 전용 폼 표시, 필수 필드 충족 시 저장 CTA 활성
- 수면: 종료 입력 시 총 수면 자동 계산, "진행 중" 토글 시 종료 비활성
- 기저귀: 유형이 대변/소변+대변일 때만 상태 칩 노출

### 빈 상태 (초기 진입)
- 날짜/시간 기본 now (성장·접종은 날짜만)
- 성장: 직전 측정값을 placeholder로 흐리게 표시(프리필 아님)
- 접종: 백신 종류 미선택, 차수 1차 기본

### 로딩 (저장 중)
- Optimistic UI: 시트 닫고 타임라인 상단 임시 카드(반투명) 삽입
- 성장 저장 시 지표 탭 그래프도 갱신 대상

### 에러
- 검증 실패: 해당 입력 인라인 에러, 저장 차단
- 저장 실패: "저장에 실패했어요" 토스트 + 롤백 + 재시도

## 7. 데이터·검증 규칙

| 입력 필드 | 필수 | 형식/범위 | 단위 | 에러 메시지 |
|-----------|------|-----------|------|-------------|
| start_at(수면) | Y | 미래 불가 | — | "미래 시간은 선택할 수 없어요" |
| end_at(수면) | N | start_at 이후 | — | "종료 시간은 시작 이후여야 해요" |
| diaper_type | Y | pee/poo/both | — | "기저귀 유형을 선택해주세요" |
| stool_state | 대변 시 Y | normal/loose/hard | — | "대변 상태를 선택해주세요" |
| height_cm | 성장 1개 Y | 0~150 | cm | "키를 0~150cm 사이로 입력해주세요" |
| weight_kg | 성장 1개 Y | 0~50 | kg | "몸무게를 0~50kg 사이로 입력해주세요" |
| head_cm | N | 0~80 | cm | "머리둘레를 0~80cm 사이로 입력해주세요" |
| photos(성장) | N | jpg/png, 각 ≤10MB, 최대 5장 | — | "이미지는 10MB 이하, 최대 5장까지 가능해요" |
| hospital_name | 병원 Y | 1~50자 | — | "병원명을 입력해주세요" |
| visit_reason | 병원 Y | enum | — | "방문 사유를 선택해주세요" |
| vaccine_code | 접종 Y | enum | — | "백신 종류를 선택해주세요" |
| dose_no | 접종 Y | 1~6/추가 | 차수 | "차수를 선택해주세요" |

> 성장: 키·몸무게·머리둘레 중 **최소 1개 이상** 입력 필수.
