## 3. 상태 분기

### 기본 (데이터 있음)
- 연결된 기기 목록 + 상태 뱃지(온/오프라인) + 새 기기 등록 카드

### 빈 상태 (데이터 없음)
- wifi-high 아이콘 + "등록된 기기가 없어요" + "새 기기 등록" CTA (ADMIN)

### 로딩 (스켈레톤)
- 기기 카드 1행 shimmer (h 96px, radius 20px)

### 에러
- 목록: "기기 정보를 불러올 수 없어요" + 재시도
- 마법사 Step1(AP 연결 실패): "기기를 찾을 수 없어요" + 재시도/도움말
- 마법사 Step2(Wi-Fi 전송 실패): "Wi-Fi 정보를 전달하지 못했어요" + 비번 확인 안내
- 마법사 Step3(서버 등록 실패): "등록에 실패했어요" + 재시도, 5GHz 경고

## 5. 권한별 차이

| 요소 | ADMIN (주양육자) | USER (보조양육자) |

| 기기 등록(마법사) | 가능 | 불가 |

| 기기 해제 | 가능 | 불가 |

> user_scenario 2-3: 기기 등록·해제는 ADMIN 전용. 설정 홈에서 USER에게 본 메뉴 미렌더링.

## 6. API 연동

### 6-1. 데이터 needs

**읽기(Read):**
| 데이터 | 필드 | 타입 | 비고 |
|--------|------|------|------|
| 기기 목록 | home_id | string | |
| 기기 항목 | device_id | string | |
| | name | string | 1~20자 |
| | status | enum(ONLINE/OFFLINE) | 뱃지 |
| | firmware_version | string | |
| | last_connected_at | datetime | 상대시간 |
| 주변 Wi-Fi | ssid_list | string[] | OS 권한 필요, 직접 입력 fallback |

**쓰기(Write):**
| 액션 | 필드 | 타입 | 검증 | 비고 |
|------|------|------|------|------|
| AP 연결 | (로컬) ap_ssid | string | 기기 AP 탐지 | 클라/OS, 서버 무관 |
| Wi-Fi 전송 | ssid | string | 필수 | 기기로 전달(로컬) |
| | password | string | 8자 이상(WPA) | 미저장(전송 후 폐기) |
| 서버 등록 | device_serial | string | 기기 제공 | ADMIN |
| | name | string | 기본값 자동 | |
| 이름 변경 | device_id, name | string | 1~20자 | ADMIN |
| 해제 | device_id | string | 확인 필요 | ADMIN |

### 6-2. 엔드포인트 매핑 (TBD)

| 액션 | METHOD /path | 비고 |
|------|--------------|------|
| 기기 목록 | `TBD` (GET /homes/{id}/devices) | |
| 서버 등록 | `TBD` (POST /homes/{id}/devices) | serial+wifi 결과, USER 403 |
| 이름 변경 | `TBD` (PATCH /devices/{id}) | USER 403 |
| 해제 | `TBD` (DELETE /devices/{id}) | USER 403 |
| AP/Wi-Fi 전달 | (로컬 통신) | 기기 AP HTTP, 서버 미경유 |

## 7. 데이터·검증 규칙

| 입력 필드 | 필수 | 형식/범위 | 단위 | 에러 메시지 |

| ssid | Y | 1자 이상 | — | "Wi-Fi 이름을 입력해주세요" |

| name | Y | 1~20자, trim | 글자 | "기기 이름을 입력해주세요" |

## 8. 네비게이션

| 구분 | 경로 |
|------|------|
| 진입 | 설정 홈 → 기기 관리 (ADMIN만 노출) / 온보딩 완료 직후 Alert [등록하기] |
| 이탈 | 마법사 완료 → 젖병/분유 연속 진행 or 목록 / 중단 → 확인 후 목록 |
| 딥링크 | `app://settings/devices` (Phase 2) |
| 뒤로가기 동작 | 마법사 진행 중엔 "등록을 취소할까요?" 확인 → 설정 홈/목록 |
