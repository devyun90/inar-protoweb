## 3. 상태 분기

### 기본 (데이터 있음)
- 마스터 ON: 알림 종류·방해금지 활성. 각 Toggle 상태 반영, ON 항목은 서브행(간격/시간대) 노출
- 마스터 OFF: 하위 섹션 전체 dim + Switch 비활성 (값은 보존)

### 빈 상태 (데이터 없음)
- 해당 없음. 최초 진입 시 기본값(수유 3h ON, 예방접종/TODO/출수 ON, 야간 22:00~07:00 ON)

### 로딩 (스켈레톤)
- 마스터 카드 + 알림 종류 카드(4행) shimmer

### 에러
- 설정 로드 실패: "알림 설정을 불러올 수 없어요" + 재시도 (로컬 우선, 서버 동기화 실패 시 토스트)
- OS 알림 권한 거부 시: 상단 배너 "기기 알림 권한이 꺼져 있어요 [설정 열기]"

## 4. 인터랙션

| 동작 | 대상 | 결과 |
|------|------|------|
| 토글 | 알림 전체 | 하위 섹션 활성/비활성, 스케줄 일괄 재계산 |
| 토글 | 수유 간격 | ON 시 간격 서브행 노출 + 다음 알림 스케줄링 |
| 탭 | 간격 서브행 | 시간 피커 시트 (1h~6h, 30분) |
| 토글 | 예방접종/TODO/출수 | 해당 알림 스케줄 on/off |
| 토글 | 야간 방해금지 | ON 시 시간대 서브행 노출 |
| 탭 | 시간대 서브행 | 시간 범위 피커 시트 |
| 탭 | 확인(피커) | 값 저장 + 관련 알림 재스케줄 |
| 탭 | 설정 열기(권한 배너) | OS 알림 설정으로 이동 |

## 5. 권한별 차이

| 요소 | ADMIN (주양육자) | USER (보조양육자) |
|------|------------------|-------------------|
| 화면 진입 | 가능 | 가능 (동일) |
| 모든 알림 토글 | 가능 | 가능 |
| 시간 설정 | 가능 | 가능 |

> 알림은 **개인 단말의 로컬 알림** 설정이므로 권한 차이 없음. 단, "기기 출수 완료"는 기기/홈에 연동된 사용자에게만 의미 있음(기기 미등록 시 행 dim + "기기 등록 후 사용 가능").

## 6. API 연동

### 6-1. 데이터 needs

> MVP는 로컬 알림(flutter_local_notifications + timezone). 설정값은 단말 로컬 저장이 기본, 다단말 동기화는 선택적 서버 저장.

**읽기(Read):**
| 데이터 | 필드 | 타입 | 비고 |
|--------|------|------|------|
| 알림 설정 | master_enabled | bool | 마스터 |
| | feeding_interval_enabled | bool | |
| | feeding_interval_minutes | int | 60~360, 30단위 |
| | vaccine_dday_enabled | bool | |
| | todo_due_enabled | bool | |
| | device_dispense_enabled | bool | |
| | dnd_enabled | bool | 야간 방해금지 |
| | dnd_start / dnd_end | time(HH:mm) | 범위 |
| 컨텍스트 | last_feeding_at | datetime | 수유 간격 스케줄 기준 |
| | device_registered | bool | 출수 행 활성 여부 |

**쓰기(Write):**
| 액션 | 필드 | 타입 | 검증 | 비고 |
|------|------|------|------|------|
| 설정 저장 | 위 설정 필드 | — | 범위 검증 | 로컬 저장 + (선택)서버 동기화 |
| 스케줄 재계산 | (로컬) | — | — | 저장 시 알림 reschedule |

### 6-2. 엔드포인트 매핑 (TBD)

| 액션 | METHOD /path | 비고 |
|------|--------------|------|

| 설정 저장 | `TBD` (PUT /me/notification-settings) | 다단말 동기화 시 |

## 7. 데이터·검증 규칙

| 입력 필드 | 필수 | 형식/범위 | 단위 | 에러 메시지 |
|-----------|------|-----------|------|-------------|
| feeding_interval_minutes | 조건부 | 60~360, 30분 단위 | 분 | "간격은 1시간~6시간으로 설정해주세요" |
| dnd_start / dnd_end | 조건부 | HH:mm, 자정 넘김 허용 | 시:분 | "시간을 선택해주세요" |
| 야간 시간대 | — | 시작=종료 불가 | — | "시작과 종료 시간이 같을 수 없어요" |

## 10. 접근성

| 항목 | 기준 |
